# 这是我的vimrc :)

## 使用方法
1. 安装vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
2. 下载vimrc文件并移动到相应文件夹
```
git clone https://github.com/SharpZhang/vimrc.git
mv ./vimrc/vimrc ~/.vimrc
```
3. 安装插件
```
vim ~/.vimrc

:PluginInstall
```
